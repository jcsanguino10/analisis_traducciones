Conjuntos coordenáveis ou equipotentes Quando falamos de conjuntos coordenáveis, comparamos conjuntos e estabelecemos relações entre eles. Veja do que se trata esse conceito. Diz-se que dois conjuntos são coordenáveis ou equipotentes quando são constituídos pelo mesmo número de elementos. maneira correta de estabelecer se dois conjuntos são coordenáveis ou não, é estabelecendo uma relação entre seus elementos. Essa relação deve ter algumas características especiais que explicamos a seguir: Relações bijetivas Imagine que temos dois conjuntos: `A` e `B.` Estabelecer uma relação entre eles é relacionar os elementos do conjunto `A` com os elementos do conjunto `B.` observa as seguintes condições: Cada elemento do conjunto `A` deve estar relacionado a um único elemento do conjunto `B.` Cada elemento do conjunto `B` deve estar relacionado a um único elemento do conjunto set `A.` Se Uma relação entre conjuntos que satisfaça essas condições é chamada de relação um-para-um. Na imagem anterior, por exemplo, cada convidado está relacionado a um único drink, e cada drink está relacionado a um único convidado. Quando é possível estabelecer uma relação um-para-um entre os conjuntos `A` e `B`, diz-se que `A` é coordenavel com `B` ou que `A` é equipotente com `B`. Caso contrário, dizemos que eles não são coordenáveis ou que não são equipotentes. Você deve ter claro que a coordenabilidade não é uma característica de um conjunto, mas uma relação entre dois conjuntos. Por exemplo, é errado dizer que o conjunto `P` é coordenável. Uma expressão apropriada seria: os conjuntos `R` e `S` são coordenáveis. Observe que se os conjuntos não tiverem o mesmo número de elementos, eles não são coordenáveis: imagine que agora uma pessoa chegou inesperadamente à festa.
