Classes de conjuntos Existem vários tipos de conjuntos que se destacam pelas suas características especiais. Conhecê-los ajudará você a entender melhor a estrutura e o mundo dos cenários. Conjunto Universal Para evitar confusão, quando definimos um conjunto devemos especificar de onde são retirados os elementos que o compõem. Isso significa que deve haver uma base de onde tiramos esses elementos, essa base sobre a qual trabalhamos é chamada de conjunto universal. Sempre usaremos a letra `U` para representar o conjunto universal. Por exemplo, se você quiser definir `B` como o conjunto que consiste nas vogais `a` e `i`, o conjunto universal pode ser o conjunto de vogais. A figura acima mostra como você pode usar diagramas de Venn para representar a relação entre o conjunto `B` e seu conjunto universal `U`. Observe que o conjunto universal pode ter exatamente os elementos dos conjuntos que abrange ou mais. Conjunto vazio Considere a existência de um conjunto que não possui elementos, é chamado de conjunto vazio. Para representar tal conjunto utilizamos o conhecido símbolo vazio, conforme a imagem abaixo: Também, fazendo uso da descrição da extensão, representamos o conjunto vazio por meio dos colchetes `{}`. Como o conjunto vazio não possui elementos, não podemos colocar nenhum elemento dentro dos colchetes. Conjuntos unitários O conjunto unitário se distingue por possuir apenas um elemento. Não importa que tipo de elemento o conjunto tenha, um gato, um cachorro, um número, uma letra ou qualquer outra coisa, se tiver apenas um elemento é chamado de conjunto unitário. Conjuntos finitos Este tipo de conjunto também se distingue pelo número de elementos que possui. Um conjunto é finito se pudermos contar o número de elementos que o compõem. Por exemplo, o conjunto de letras na língua espanhola é finito porque existem `27` letras no total. Outros conjuntos finitos são mostrados na imagem à direita. Você pode perceber que conjuntos unitários também são finitos. Conjuntos infinitos Não é fácil encontrar exemplos deste tipo de conjuntos na natureza. Conjuntos infinitos são aqueles aos quais não podemos contar o número de elementos que os compõem. O método mais fácil de representar este tipo de conjuntos é por compreensão. Basta mencionar as características que os elementos do conjunto têm em comum e estaremos determinando todas elas. Considere o conjunto de números que terminam em três, poderíamos defini-lo assim: Seja `T={x|x\ é\ número\ y\ ter\m\i\na\ em\ três}`. Há também uma maneira de representar alguns conjuntos infinitos por extensão. Basta exibir os primeiros elementos do conjunto e indicar com reticências que a lista continua indefinidamente. No caso do conjunto `T`, definido no parágrafo anterior e formado pelos números que terminam em três, temos `T={3, 13, 23, 33, 43, 53, ...}`. Os exemplos mais simples e comuns de conjuntos infinitos são encontrados em números. Quantos números pares existem? Quantos múltiplos tem três? Esses conjuntos são infinitos, e não é porque está além de nossa capacidade de contar o número de elementos que eles possuem. É que é impossível fazê-lo porque não existe um número que represente o número de elementos que o conjunto contém. Você não deve confundir conjuntos infinitos com conjuntos finitos que possuem um grande número de elementos. Por exemplo, você considera o conjunto de todos os grãos de areia do planeta Terra um conjunto infinito? Nesse caso, mesmo que o conjunto tenha um grande número de elementos, deve haver um número que o represente, mesmo que seja muito grande.
