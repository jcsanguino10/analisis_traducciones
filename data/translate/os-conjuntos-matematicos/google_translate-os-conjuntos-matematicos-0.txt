O que é um conjunto? Para que você entenda o maravilhoso mundo da matemática, você deve partir do mais básico, do começo de tudo: os conjuntos. Antes que o homem entendesse o conceito de número, ele deve ter entendido de onde eles vieram e o que eles representavam. Portanto, a ideia de número segue o entendimento de conjuntos. Você colecionou fichas, brinquedos ou fotos para um álbum? Imagine que os conjuntos são exatamente isso, uma coleção de objetos que podem ser classificados pelas características que têm em comum (fichas, folhas, etc). Um elemento é... Chamamos os objetos que compõem os conjuntos de elementos. Um requisito fundamental para que um agrupamento de objetos seja chamado de conjunto é que seja possível determinar se um objeto específico pertence ou não a ele. Por exemplo, o grupo de coisas belas não é um conjunto, pois haverá coisas que são belas para alguns, mas não para outros. Neste caso diz-se que o conjunto não está bem definido. Por outro lado, se pensarmos no conjunto dos planetas do sistema solar, os elementos desse conjunto serão justamente Mercúrio, Vênus, Terra, Marte, Júpiter, Saturno, Urano e Netuno. Este conjunto está bem definido. Representação gráfica de conjuntos: Diagramas de Venn Para representar conjuntos graficamente, podem ser utilizados diagramas de Venn. Este método consiste em representar os conjuntos por meio de círculos e desenhar dentro deles os elementos que os compõem.
