Decimais para medir Quando uma medição é feita, o resultado pode ter partes unitárias, e essas partes podem ser representadas com frações ou decimais. Observe o seguinte exemplo: Nas instruções de uma receita diz que para preparar um bolo deve-se usar `1,5` xícaras de farinha. O que isto significa? Para entender melhor este exemplo, faremos uma comparação com a reta numérica. Na linha, a unidade de medida é a distância entre `0` e `1`, no exemplo a unidade de medida é um copo. Substitua a palavra unidade pela palavra xícara para facilitar as coisas: uma unidade é uma xícara, duas unidades são duas xícaras, etc: Como `1,5` é uma unidade e meia, `1,5` xícaras É uma xícara e meia. Se as unidades `1,5` estiverem localizadas, descobriremos que seu lugar na linha está bem no meio do um e do dois. Portanto, xícaras de &#39;1,5&#39; devem ser representadas no mesmo local. Observe que o número `1,5` é equivalente a `3/2`. Portanto, se você deseja expressar essa medida em frações, pode dizer que &quot;1,5&quot; xícara é o mesmo que três meias xícaras: &quot;3/2&quot; xícara.
