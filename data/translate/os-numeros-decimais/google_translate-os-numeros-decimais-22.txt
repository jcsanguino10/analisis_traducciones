Decimais e conversão de moedas Pesos, dólares, euros ou reais são alguns dos tipos de moedas que podemos encontrar no mundo. Se você for viajar ou pretende fazer uma compra em outro país, será útil converter de um para o outro. Observe: Laura sai de férias da Colômbia para a Itália com `12.500.000` pesos. Quantos euros Laura tem para gastar na Itália? Para responder a esta pergunta é necessário saber quantos pesos equivalem a um euro. Suponha que esse preço seja de `3300` pesos por cada euro. Agora imagine que vão distribuir os `12.500.000` pesos` em pequenos grupos de `3.300`. Cada um destes pacotes representa um euro, pelo que o número de pacotes que podem ser formados será o número de euros equivalente a `12&#39;500.000`. Lembre-se que a operação que nos permite fazer esses agrupamentos é a divisão. Para saber quantos grupos podemos formar devemos realizar a operação `12&#39;500.000 div 3300`. O resultado disso é `3787, compasso 87`. Arredondando este valor para duas casas decimais, obtemos que Laura tem `3787,88` euros para suas férias. Outro exemplo Steve planeja fazer uma temporada de estudos no Brasil e tem `7.500` dólares para isso. Quantos reais Steve pode obter com o dinheiro que tem, se cada dólar equivale a `3,33` reais? Para visualizar a situação, na imagem à esquerda os dólares foram representados com uma nota verde e os reais com moedas. Observe que R$ 3,33 foram colocados ao lado de cada nota. Como Steve tem `7.500` dólares, para calcular quantos reais ele tem, ele deve somar o valor `3,33` sete mil e quinhentas vezes. Para isso, é realizada a operação `7.500xx3.33`, que resulta em `24.975`. Pode-se então dizer que Steve tem `24.975` reais. Resumindo Observe que para resolver essas situações, no primeiro exemplo foi dividido, enquanto no segundo foi multiplicado. A razão para isso é que no primeiro exemplo você estava saindo de uma moeda de menor valor para uma de maior valor: de pesos colombianos para euros. Por sua vez, no segundo exemplo, uma moeda de maior valor foi passada para outra de menor valor: de dólares para reais. Para não ficar confuso, lembre-se: Para converter de uma moeda de menor valor para uma de maior valor, você deve dividir. Para converter de uma moeda de maior valor para uma de menor valor, você deve multiplicar.
