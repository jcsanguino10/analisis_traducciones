DreamSpark: O que é e como funciona Se o seu sonho é criar o próximo avanço tecnológico, o DreamSpark oferece as ferramentas para que você fique um passo mais perto de torná-lo realidade! Atualmente, a Microsoft possui diversos projetos que buscam incentivar a experiência e a pesquisa dos interessados em tecnologia e programação. Um desses programas é chamado DreamSpark for Students, uma plataforma onde você pode baixar gratuitamente ferramentas profissionais de design e desenvolvimento. Dessa forma você pode experimentar em busca de um avanço tecnológico ou ter uma certa vantagem em sua carreira profissional. Observe que para acessar este tipo de software gratuitamente, você deve provar que ainda pertence ou está vinculado a uma entidade educacional. Como acessar os programas oferecidos no DreamSpark? Para adquirir esses softwares que a Microsoft oferece gratuitamente, você deve inserir o seguinte link no navegador: Dreamspark. Caso tenha problemas com o idioma da página, você pode alterá-lo. Basta clicar na seta à direita da opção Idioma, localizada no canto superior direito da página. Lá, será exibido um menu com todos os idiomas disponíveis. Escolha aquele com o qual você se sente mais seguro. Agora, na parte superior da tela, clique na opção Alunos para realizar o cadastro. Lembre-se de que, ao concluir o processo de inscrição, você deve validar na página que ainda pertence a uma entidade educacional. Para fazer isso, a Microsoft solicitará qualquer uma das seguintes informações: O endereço de e-mail da faculdade ou universidade. A credencial ou cartão que o identifica como aluno. Código de verificação fornecido pela Microsoft à instituição à qual você pertence. Um cartão de identificação de estudante internacional. Forneça qualquer tipo de identificação que o credencie como aluno (certificados de estudante). Depois que seu cadastro for aprovado, você poderá começar a baixar os programas gratuitos oferecidos pela Microsoft para estudantes. O melhor de tudo é que eles vêm com as licenças incluídas. Alguns aplicativos que você pode acessar como alunos são: Visual Studio: Você pode encontrar este programa desde sua versão 2010 até a Comunidade 2013. É um ambiente de desenvolvimento integrado ou Ambiente de Desenvolvimento Integrado (IDE), programa ou aplicativo formado por um conjunto de ferramentas que facilitam a edição de código no desenvolvimento de software. O Visual Studio permite que você desenvolva aplicativos Web, aplicativos ou programas de desktop e consoles de comando. Na versão mais recente deste aplicativo, você pode programar em diferentes linguagens, como .NET, Pyton, C++, HTML, Javascripts. Além disso, este ambiente ou programa é bastante amigável e contém diversas ferramentas para te auxiliar no processo de desenvolvimento. Windows Server Você encontra da versão 2008 a 2012 R2. É uma versão servidor do Windows, possui uma interface amigável e pode ser instalado tanto em máquinas físicas quanto virtuais. Neste servidor você pode publicar todas as aplicações web que você desenvolveu. Além disso, oferece todas as funcionalidades de um servidor para armazenar arquivos. O SQL Server é um sistema de banco de dados. Nele, você pode salvar ou armazenar os dados necessários para os aplicativos que você desenvolve: contas de usuários, registros, arquivos, entre outros dados. Sua interface é intuitiva e prática, para que você possa facilmente configurar e criar os bancos de dados que você precisa.
