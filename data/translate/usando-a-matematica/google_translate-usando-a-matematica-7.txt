Números binários Um número binário é uma representação alternativa de um número usando apenas dois símbolos. Números decimais ou o sistema de numeração decimal Na vida cotidiana usamos o sistema de numeração decimal, no qual os números são representados por meio de dez símbolos diferentes que conhecemos como os dígitos de `0` a `9.` Podemos escrever diferentes quantidades usando um combinação destes símbolos em posições diferentes, como: `3847, 145` ou `18.` Números binários No sistema binário, os números são representados usando apenas dois símbolos diferentes que são `0` e `1.` Podemos escrever quantidades diferentes usando uma combinação desses símbolos em diferentes posições, como: `1011110, 1111` ou `100`. A “base” Em geral, podemos escolher qualquer número de símbolos e ainda podemos representar qualquer número. Essa quantidade é chamada de base do sistema numérico. Então, quando você ouve sistema na base dez, sabe que se refere ao sistema de numeração decimal e quando ouve sistema na base dois, sabe que se refere ao sistema de numeração binário. Existem vários sistemas numéricos com bases diferentes, os mais utilizados são binário (base 2), octal (base 8), decimal (base 10) e hexadecimal (base 16). Para que isso é usado? O sistema binário é a base da tecnologia digital, qualquer dispositivo que possua circuitos integrados (chips) é possível graças a este sistema numérico. Sendo o sistema binário o sistema de numeração mais simples, uma vez que utiliza apenas dois símbolos, é possível armazenar e manipular números de forma física, por exemplo como a eletricidade que passa por um fio, como a luz que passa por uma fibra ótica, como campos magnéticos em um disco rígido ou como sinais eletromagnéticos viajando pelo ar. Graças a essa forma física dos números binários, é possível que eles existam no microprocessador, na memória ou no disco rígido do seu computador ou que viajem pela Internet ou rede Wi-Fi. A coisa mais útil sobre o sistema numérico binário é que ele pode ser materializado com poucos elementos físicos. Mas os computadores podem fazer muitas coisas, não apenas com números... ...bem, na verdade... não. Tudo o que seu computador ou smartphone pode fazer é possível porque algo foi digitalizado, ou seja, convertido em uma sequência de números binários. Cada letra do seu documento do word, a música que você ouve no soundcloud, a foto que você vê no facebook, a voz do seu amigo no skype e a carinha sorridente que você envia no whatsapp são finalmente uma sequência de números binários fisicamente representados.
